# #57 Photo Privacy Cleaner

A minimalist, high-performance web tool designed to strip Exif metadata (GPS, camera info, timestamps) from your photos to protect your digital privacy.

## Core Philosophy
* **Privacy First**: All processing happens locally in your browser. Your photos are never uploaded to any server.
* **1 Tool / 1 Action**: Focuses exclusively on removing invisible tracking data while preserving visual quality.
* **Frictionless**: No login, no ads, no tracking. Just drop and clean.

## Key Features
* **Metadata Stripping**: Automatically removes GPS coordinates, device models, and exposure settings.
* **Bulk Processing**: Support for selecting and cleaning multiple photos simultaneously.
* **Privacy Renaming**: Automatically renames files to `safe_photo_[timestamp].jpg` to eliminate any traces in the filename.
* **iOS Optimized**: Stable batch downloading specifically tuned for mobile Safari.

## How to Use
1. **Select**: Tap the upload zone to choose one or more photos.
2. **Process**: The tool instantly recreates the images on a local canvas, leaving metadata behind.
3. **Download**: Tap the download button. For multiple photos, the tool will trigger consecutive downloads automatically.

## Design System
* **Background**: #fff
* **Main Color**: #333
* **Border Radius**: 15px (Consistent with Fridge Combo Constitution)
* **Performance**: Optimized for low-spec mobile and desktop browsers.

## Privacy Policy
* No login required.
* No tracking scripts.
* No advertisements.
* Browser-only execution.
